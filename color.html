  1. By default the Hexadecimal Colour code is placed at White.
  2. Drag the slider of each colour element which incudes Red, Green and Blue.
  3. The background colour will change accordingly to the precision of the colour in the background.
  4. Copy the code generated below and add it to your css project file to add colours of your choice!
  5. Dont worry about the font colour of the Rules written they are programed to switch their colours if the background goes dark.
